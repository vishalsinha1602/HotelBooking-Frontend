🏨 HotelBooking-Backend

A production-ready Spring Boot backend for a Hotel Booking system built with modern architecture, clean code practices, and scalable design.

🚀 Tech Stack
⚙️ Spring Boot
🌐 Spring MVC (REST APIs)
🗄️ Spring Data JPA (Hibernate)
🔐 Spring Security
🧪 JUnit & Testing
🐳 Docker (Upcoming)
☁️ CI/CD (Upcoming)
📌 Project Overview

This project is a backend system for hotel booking, designed to simulate a real-world production application.

It covers:

User management 👤
Hotel & room management 🏨
Booking system 📅
Secure APIs 🔐
Scalable architecture 🧱

⚙️ How to Run
# Clone the repo
git clone https://github.com/your-username/HotelBooking-Backend.git

# Navigate to project
cd HotelBooking-Backend

# Run the application
./mvnw spring-boot:run
🧪 Running Tests
./mvnw test
